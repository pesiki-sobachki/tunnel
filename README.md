# Простой и надежный обратный SSH-туннель

Этот проект предоставляет простой и надежный скрипт для открытия доступа к сервису, запущенному на вашей локальной машине, через удаленный сервер. Он использует утилиту `autossh`, чтобы гарантировать автоматическое восстановление туннеля в случае обрыва соединения.

Это минималистичное решение без зависимостей для разработчиков, системных администраторов и энтузиастов, которым нужен быстрый и безопасный способ создать публичную точку доступа для локального приложения.

## Ключевые особенности

*   **Простота:** Один скрипт с подробными комментариями. Никакой сложной настройки.
*   **Безопасность:** Использует стандартное, проверенное временем шифрование и аутентификацию по SSH-ключам. Включает инструкцию по созданию выделенного пользователя с сильно ограниченными правами.
*   **Надежность:** Использует `autossh` для мониторинга соединения и автоматического переподключения в случае сбоя.
*   **Универсальность:** Может пробрасывать любой TCP-трафик, а не только HTTP. Полезно для веб-серверов, баз данных, игровых серверов, IoT-устройств и многого другого.

## Как это работает

Скрипт устанавливает обратный SSH-туннель с вашей локальной машины на удаленный сервер. Это означает, что удаленный сервер начинает слушать публичный порт и перенаправляет весь входящий трафик через зашифрованный туннель на указанный порт вашей локальной машины.

```text
                  <-- Интернет-трафик -->
                         |
+-----------------------------------------------------------+
|                      Удаленный сервер                     |
|                                                           |
|    Файрвол          +-----------------+                   |
|       |             |   Порт :8080    |                   |
|    (Порт 80) <----->| (или любой другой) |                |
+---------------------|-----------------|-------------------+
                      |      ^
                      |      | (Весь трафик перенаправляется)
                      v      |
+-----------------------------------------------------------+
|            Обратный SSH-туннель (Зашифровано)             |
+-----------------------------------------------------------+
                      |      ^
                      |      | (Соединение инициируется отсюда)
                      v      |
+-----------------------------------------------------------+
|                      Локальная машина                     |
|                                                           |
|    +------------------------------------------------+     |
|    | Ваше приложение (веб-сервер, БД и т.д.)        |     |
|    | Слушает на localhost:3000 (или любом другом)   |     |
|    +------------------------------------------------+     |
+-----------------------------------------------------------+
```

## Частые сценарии использования

*   **Веб-разработка:** Безопасно продемонстрировать веб-приложение, запущенное на вашем ноутбуке, клиенту без необходимости его развертывания.
*   **Доступ к базам данных:** Предоставить временный и безопасный доступ к локальной базе данных для удаленного коллеги.
*   **IoT и Raspberry Pi:** Получить доступ к устройству в вашей домашней сети из любой точки мира.
*   **Тестирование веб-хуков:** Получать и тестировать веб-хуки от сервисов вроде Stripe или GitHub в вашем локальном окружении.
